# SwatLLM Commands Syntax

This is the syntax for the SLLM.

## SLLM format (*.sllm)

```
{team color} {instruction} {arg0} {arg1} {arg2}

{instruction} {arg0} {arg1} {arg2}

### mov team to s. pos
mov {TEAM} {id}

### cover pos
cov {TEAM} {id}

### breach to s. pos
bre {TEAM} {id}

### stack up
sup {TEAM} {id}

### fall in
fin {TEAM} {id}

### search and secure
sas {TEAM} {id}

### hold
hod {TEAM} {id}

### RESTRAIN
ret {TEAM} {id}

### yell
yel {TEAM} {id}

```
